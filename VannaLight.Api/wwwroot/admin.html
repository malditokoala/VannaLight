Cómo usar este wizard
Sigue este orden para evitar errores y dejar el dominio listo en una sola pasada.
1. Guarda el workspace con su conexión.
2. Descubre el schema y elige las tablas que sí vas a permitir.
3. Prepara el dominio para generar contexto técnico.
4. Ejecuta una pregunta real para validar el resultado.
